## Android NFC Toggler    
####  Two ways to Enable/Disable NFC     
1. Widget     
2. QuickSetting     
Needs ROOT    

#### Environment    
1. Android 7.0 or above    
2. Android Studio 3.1.4 or above
3. Kotlin and java Code    

#### Known Bugs
